# Projeto de Ciência de Dados

Este repositório apresenta as subdivisões de um projeto completo desenvolvido para a disciplina de **Ciência de Dados**. O objetivo principal é perpassar por todas as etapas fundamentais do processo de **Mineração de Dados (KDD)**, aplicando conceitos teóricos e práticos de forma sequencial.

O pipeline do projeto foi estruturado nas seguintes fases:

* **Escolha e Entendimento da Base de Dados**
* **Análise Exploratória de Dados (EDA)**
* **Pré-processamento e Limpeza de Dados**
* **Análise Descritiva**
* **Agrupamento (Clustering)**
* **Classificação (Machine Learning)**

---

## Abordagem Metodológica

Em cada uma das etapas listadas, o projeto busca evidenciar três pilares essenciais:

1. **Problemas Relacionados:** Identificação e diagnóstico dos desafios inerentes aos dados (como desbalanceamento de classes, multicolinearidade, possíveis dados ausentes, falta de relação linear entre atributos, etc.).
2. **Algoritmos e Métodos:** Aplicação prática de técnicas computacionais para a resolução dos problemas identificados (ex: PCA, MinMaxScaler, Correlação de Pearson, algoritmos de classificação, etc.).
3. **Fundamentação Teórica:** Justificativa científica e conceitual por trás de cada tomada de decisão, baseando-se estritamente nas aulas vistas e no livro norteador da disciplina.
